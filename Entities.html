HTML Entities

Here are some commonly used HTML entities:

  • & : & (Ampersand)
  • < : < (Less than)
  • > : > (Greater than)
  • " : " (Double Quote)
  • ' : ' (Single Quote)
  • ¢ : ¢ (Cent)
  • £ : £ (Pound)
  • ¥ : ¥ (Yen)
  • € : € (Euro)
  • © : © (Copyright)
  • ® : ® (Registered)
  • ™ : ™ (Trademark)
  • ° : ° (Degree)
  • ± : ± (Plus-minus)
